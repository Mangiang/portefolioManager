# PortefolioManager

This is a Qt based manager for my portfolio.
<a href="https://arthur-joly.fr">https://arthur-joly.fr</a>

### Documentation

The documentation is generated by Doxygen based on the ./Doxygen/Doxyfile file.
<div class="warning">
**WARNING**: The dot tool from **GraphViz is needed** to display the diagrams.
</div>



Usage: 
~~~~~~~~~~~~~~~{.sh}
doxygen ./Doxygen/Doxyfile
~~~~~~~~~~~~~~~
